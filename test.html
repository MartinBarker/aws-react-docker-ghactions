Changelog:

  • 10/25/2022: Added new checkbox 'Check box to use URL input song titles with file timestamps' which will appear when a Discogs URL is used as input. If selected, the next time you drag/choose files, it will use the song title values from the discogs url input, but use the timestamps from your files.
  • 04/17/2021: Added loading spinner when tags are being generated, changed artist tags generation system so that tags for each artist are organized smarter.
  • 11/18/2020: tagger.site now accepts a Discogs URL as input by appending the Discogs URL after tagger.site. So If you visit https://tagger.site/https://www.discogs.com/New-Golden-Wings-Its-Rough-Its-Tough/release/5390422 it will have the Discogs data already generated when the page loads.
  • 11/17/2020: .cue files now work for generating a timestamped tracklist, and I improved the error handling for release URLs like this which have a broken artist page.

Roadmap:

  • 10/25/2022: Currently working on a react.js rewrite, as well as adding tons of new features.
